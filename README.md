Поиск фильмов по описанию на естественном языке с использованием sentence-transformers и FAISS.

## Возможности

- 🔍 Семантический поиск на русском языке
- 📊 Данные из Кинопоиска (2000 фильмов)
- 🤖 Мультиязычная модель `intfloat/multilingual-e5-base`
- ⚡ Быстрый поиск через FAISS
- 💾 Сохранение и загрузка индекса
- 🎬 Рейтинги, постеры, жанры, длительность
- 🌐 Веб-интерфейс в стиле маркетплейса

## 📈 Установка

```bash
pip install -r requirements.txt
```

## ✅ Индексация

```bash
python build_index.py
```

## 🔃 Зависимости

- `sentence-transformers` — кодирование текста в векторы
- `faiss-cpu` — быстрый поиск ближайших соседей
- `fastapi` + `uvicorn` — веб-фреймворк
- `numpy` — работа с векторами

## ❓ Как это работает

1. **Кодирование**: Названия и описания фильмов преобразуются в векторы (эмбеддинги)
2. **Индексация**: Векторы сохраняются в FAISS для быстрого поиска
3. **Поиск**: Запрос пользователя кодируется и находится ближайший фильм по косинусному сходству

## 💾 Данные

Датасет из Кинопоиска содержит:
- 2000 фильмов
- Описания на русском языке
- Рейтинги, жанры, годы выпуска
- Постеры фильмов
- Длительность фильмов

## API Endpoints

- `GET /` — Главная страница
- `POST /api/search` — Поиск фильмов (`{"query": "...", "top_k": 20}`)
- `GET /api/film/{id}` — Информация о фильме
- `GET /api/popular` — Популярные фильмы по рейтингу
- `GET /api/genres` — Список жанров
- `GET /api/genre/{name}` — Фильмы по жанру
